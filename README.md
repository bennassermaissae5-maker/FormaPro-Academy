# 🎓 FormaPro Academy – Système de Gestion de Centre de Formation

Ce projet est une plateforme web fullstack permettant de gérer un centre de formation de manière dynamique. Il inclut une interface publique pour les stagiaires et un tableau de bord pour l'administration.

## 🚀 Fonctionnalités clés
- **Catalogue de formations :** Affichage dynamique avec filtres par catégorie et niveau.
- **Recherche temps réel :** Système de recherche optimisé en JavaScript.
- **Gestion des inscriptions :** Espace stagiaire avec historique des inscriptions et statuts de paiement.
- **Sécurité :** Hachage des mots de passe avec Bcrypt et protection contre les injections SQL (requêtes préparées PDO).
- **Dashboard Admin :** Statistiques en temps réel sur les formations et les revenus.

## 🛠️ Technologies utilisées
- **Frontend :** HTML5, CSS3 (Modern UI), JavaScript (AJAX/Fetch API).
- **Backend :** PHP 8.1 (Architecture orientée API).
- **Base de données :** MySQL (Modélisation relationnelle complète).

## 📂 Structure du projet
- `/sql` : Contient le schéma de la base de données (`database.sql`).
- `/php` : Logique serveur et API REST.
- `/admin` : Interface de gestion administrative.
- `/js` & `/css` : Scripts d'interactivité et design responsive.
